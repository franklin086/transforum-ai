# TransForum AI Alpha 1.0.1 Demo Checklist

## 演示前检查

- Git 已同步
- Whisper 模型 Ready
- 麦克风可用
- 投影仪连接
- 后端启动
- 前端启动
- 浏览器可访问
- `/api/health` 返回 Alpha 1.0.1

## 演示流程

1. 打开首页
2. 点击 Start First Real Meeting Demo
3. 创建会议
4. 点击 Start Realtime Caption
5. 打开 Screen Mode
6. 朗读测试语句
7. 检查中英双语字幕
8. 结束会议
9. 生成会议纪要
10. 打开会议纪要页
